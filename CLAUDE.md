# Tomme du Périgord — Project Instructions

## Deployment
- Hosted on **GitHub Pages** — pushing to `main` automatically deploys to https://tomme-perigord.fr
- There is NO Vercel, no build step. Just commit and push.

## Stack
- Static site: HTML, CSS, JS (no framework, no bundler)
- Single shared stylesheet: `css/style.css`
- Single shared script: `js/main.js`
